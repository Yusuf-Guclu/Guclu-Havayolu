# Güçlü Havayolu – Uçuş ve Bilet Yönetim Sistemi

Güçlü Havayolu, ASP.NET Core kullanılarak geliştirilmiş bir uçuş ve bilet yönetim sistemidir.  
Kullanıcılar site üzerinden uçuşları görüntüleyebilir, giriş yaparak bilet satın alabilir ve kendi biletlerini görüntüleyebilirler.  
Admin paneli üzerinden ise uçuş ekleme, düzenleme ve silme işlemleri yapılabilir.

## Özellikler
Kullanıcı kayıt ve giriş sistemi (manuel session yönetimi)  
Giriş yapan kullanıcılar için bilet alma işlemi  
Biletlerin veritabanına kaydedilmesi (SQL Server)  
Admin tarafından uçuş ekleme, silme ve güncelleme  
Ödeme sayfası ve kart ekleme özelliği  
Responsive (mobil uyumlu) frontend  
Tailwind CSS ve Bootstrap tabanlı tasarım  

## Kullanılan Teknolojiler
ASP.NET Core MVC  
SQL Server Management Studio  
Entity Framework Core  
Tailwind CSS / SCSS  
HTML, Bootstrap, JavaScript
